To Run:

    server.js

to build:

    builder.js
    
__this builds build/client.js__    

to Run build:

    server-build.js
    
to configure NON build 
    
    config.js

to configure build 
    
    config-build.js


this also works in node-webkit!
-------------------------------

to switch to build vs dev. just toggle the isBuild option in config-build.js.
